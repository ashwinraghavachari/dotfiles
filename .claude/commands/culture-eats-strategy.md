# Culture Eats Strategy

When user instructions conflict with existing codebase patterns:
1. State the conflict explicitly: "You asked for X, but the codebase uses Y"
2. Show a concrete example of the existing pattern
3. Ask: "Should I match existing patterns or follow your instruction?"
4. If following new instruction, ask if they want the existing code refactored for consistency
