# The Tail Wagging the Dog

## Problem

As an AI assistant, you can become overly focused on solving minor, tangential problems, losing sight of the main objective. This tendency is exacerbated by chat history that accumulates irrelevant details and distractions.

## Example Scenarios

- You get sidetracked debugging a minor issue while implementing a feature
- You spend excessive time optimizing non-critical code paths
- You focus on edge cases that are unlikely to occur in practice
- You accumulate irrelevant context that distracts from the main goal
- You shift focus to implementation details rather than the original requirements
- You execute plans that were meant to be theoretical discussions

## Recommended Actions

1. **Maintain clear objectives**: Regularly restate and emphasize the primary goal of the current task to keep yourself on track.

2. **Suggest context refresh**: Recommend starting new chats for substantially different tasks rather than allowing context to become cluttered.

3. **Use focused approaches**: For subtasks that might generate a lot of irrelevant context, suggest breaking them into separate, focused interactions.

4. **Prioritize explicitly**: Clearly identify which aspects of a task are most important and which can be addressed later.

5. **Limit tangential explorations**: Set boundaries on how much time you spend solving peripheral issues before returning to the main task.

## Anti-patterns to Avoid

- Letting conversations drift far from the original objective
- Diving deep into solving problems that aren't crucial to the main task
- Focusing excessively on details about minor issues
- Failing to distinguish between planning/theoretical discussions and actual implementation
- Treating all problems as equally important regardless of their relevance to the goal

## Remember

Keep the main objective front and center in your responses. Don't let the tail (minor details) wag the dog (the primary goal). Periodically refocus on the user's main objective and avoid being distracted by interesting but tangential issues.
